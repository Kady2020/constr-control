NPM - менеджер пакетов

Для запуска NPM в проекте: устанавливаем NODE.js,
открываем редактор или терминал от имени администратора,
пишем команду Set-ExecutionPolicy RemoteSigned,
далее устанавливаем галп глобально npm i gulp-cli -g

В редакторе открываем папку проекта и в терминале
инициализирум файл настроек npm init, далее устанавливаем галп
локально npm i gulp -D (флаг -D позволяет сохранять версию установленных пакетов)

Убедимся, что всё работает корректно при помощи команда gulp -v
В результате мы должны увидеть версию CLI version и Local version
и если всё работает верно, то переходим к написанию задач создав в корне
файл gulpfile.js

===============================================

npm i - установка готовых пакетов
@img/ - подключение изображений из разных директорий
npm run svgSprite - создание спрайтов
npm run dev (gulp dev) - режим разработчика
npm run build (gulp build) - режим продакшена
npm run zip - создание архива продакшена
npm run deploy - выгрузка проекта ftp
